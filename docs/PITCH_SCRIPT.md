- [ ] Open app in Chrome full-screen (hide bookmarks bar)
- [ ] Start in Demo mode so signals are pre-loaded
- [ ] Pre-type "Analyze Zomato" — don't waste time typing live
- [ ] Show the 4-step routing animation in real time
- [ ] Click TITAN signal card to show StockInfoDrawer
- [ ] Click "Detect Patterns" on RELIANCE chart

### Suggested B-Roll Moments
- Topbar with NIFTY/SENSEX scrolling live *(0:20)*
- AI routing panel with Groq → Claude → Groq labels *(0:50–1:20)*
- Signal cards with Nivesh Scores *(1:30)*
- Pattern card appearing below chart *(1:50)*
- Split-second: mobile view (if responsive) *(2:40)*

### Voiceover Tips
- Speak at 160 words/minute — slightly faster than conversational
- Pause 0.5s after each step completes on screen
- Emphasize: "90 seconds", "₹2,600 Crore", "89% reduction"
- End on a confident, declarative tone — don't rush the close

### Tools for Recording
- **Mac:** QuickTime Player (built-in) or Loom
- **Screen + Webcam:** Loom (free tier works)
- **Editing:** CapCut (free) or iMovie
- **Captions:** Add auto-captions in CapCut for accessibility
